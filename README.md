# SP_500 APP

## Overview
This project focuses on visualizing companies in the S&P 500 stock data and trading insights plus the ability to forecast those stocks.

## Features
- Visualizing stock data
- Visualizing trading insights
- Forecasting

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Sarima
- Prophet
- Jupyter Notebook
- yfinance
